About the Project

This project generates descriptive captions for uploaded images using a deep learning pipeline. It employs the InceptionV3 model for image feature extraction and a Recurrent Neural Network (LSTM) for sequence generation.

The model is trained on a large dataset of images and corresponding captions. Beam search is implemented to generate the most relevant and high-quality captions. The workflow combines image processing (using Convolutional Neural Networks) and natural language processing (sequence prediction).

This project demonstrates the synergy of computer vision and natural language processing, enabling machines to "see" and "describe" images in a human-like manner. It was developed as part of the ECE5831 Fall 2024 course under the guidance of Professor Jaerok Kwon.

Tools used include TensorFlow/Keras for deep learning, Flask for the web interface, and Bootstrap for responsive design.
